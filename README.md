# Triangle-Perimeter
I made a triangle perimeter calculator using JavaScript. You enter the coordinates for three points (A, B, and C) into input boxes on the page and then click a button to run it. When the button is clicked, a function takes the values from the inputs and converts them into numbers so they can be used in calculations.

Then I use another function to calculate the distance between two points using the distance formula. I reuse that function to find all three side lengths (AB, BC, and CA). After that, I display each side length on the page and add them together to get the total perimeter of the triangle.

It’s a simple project, but it helped me practice working with user input, functions, and updating the page with JavaScript.
