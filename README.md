# Triangle-Perimeter
This program is a triangle perimeter calculator I made using JavaScript and coordinate geometry. I connected a button to a function that runs when clicked, which takes user input for the coordinates of three points (A, B, and C) from text boxes and converts them into numbers. Then, I used a separate function to calculate the distance between each pair of points using the distance formula, and displayed those side lengths on the page. Finally, I added the three distances together to find and display the total perimeter of the triangle, making it a simple tool that combines user input, math, and DOM manipulation.
